# Profiles Extension

Adds profile modes for thinking + tool activation.

Canonical profile names:
- `ultrathink`
- `execute`
- `ship`
- `fast`

Friendly aliases:
- `meta`, `deep`, `think` → `ultrathink`
- `build`, `dev`, `workhorse` → `execute`
- `release`, `deliver` → `ship`
- `quick` → `fast`

Use:
- `/profile <name>`
- `/profile list`
- `pi --profile <name>`
- `PI_DEFAULT_PROFILE=<name>`
