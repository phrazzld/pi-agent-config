# daybook extension

One-on-one journaling posture optimized for charisma and reflection quality.

## What it adds

- Daybook system-prompt overlay on every run
- Tone control command:
  - `/daybook-tone charismatic|calm|coach`
- Quick conversation starter:
  - `/daybook-kickoff`
- Session-persistent tone state via custom entries

## Design intent

- Prioritize writing quality and engagement over coding workflow orchestration
- Encourage thoughtful follow-up questions
- Use tools selectively for factual grounding and memory recall
